# Projeto Orientado a Objeto - Gerenciamento de Usuários

## Descrição
Este projeto consiste em uma aplicação web com frontend e backend integrados para cadastro e listagem de usuários.

## Arquitetura
O sistema foi desenvolvido seguindo os princípios de orientação a objeto e organização em camadas:

- Model: representa a entidade User
- Repository: responsável pela leitura e escrita dos dados
- Controller: responsável pelas regras de negócio e respostas da API
- View: interface web disponível na pasta public

## Funcionalidades
- Cadastrar usuário
- Listar usuários cadastrados
- Buscar usuário por id
- Atualizar usuário
- Remover usuário

## Tecnologias
- Node.js
- Express
- JavaScript
- HTML
- CSS
